# RADICE | Luxury Hospitality Interface (Portfolio)

High-performance, artisan-grade landing page built for modern culinary brands. This project demonstrates high-level mastery of Frontend Architecture, SEO, and UX Design.

## 🚀 Why Clients Hire Me
- **Zero-Dependency Code:** No heavy libraries like React or Bootstrap. Just pure, lightning-fast performance.
- **Conversion Focused:** Designed with clear CTAs (Call to Action) to turn visitors into bookings.
- **A11y & SEO:** Built with semantic HTML5 and JSON-LD schema so Google understands the business immediately.
- **Micro-Interactions:** Smooth GPU-accelerated animations that provide a premium "feel."

## 🛠️ Performance Architecture
- **CSS Grid/Flexbox:** Modern layout system for 100% responsiveness.
- **Intersection Observer:** Lazy-loading logic for scroll-triggered animations.
- **Mobile-First:** Clean navigation system with scroll-lock state management.
